# X1(32) C Root-Existence Benchmark

Date: 2026-06-02 PDT

Purpose: close the remaining cost-floor question for direct X1(32):

```text
If we sample Sutherland's optimized X1(32) model by random x-fibers, how
expensive is it merely to decide whether the degree-10 fiber polynomial in y
has an Fp root?
```

This is a lower bound on direct X1(32) production cost. A full sampler would
still need:

```text
1. extracting the roots,
2. mapping optimized X1 coordinates to Tate normal form,
3. converting the marked point to Montgomery x-only form,
4. applying nonsplit or other filters,
5. running the remaining halving chain.
```

degree_x = 11

## Implementation

Added research-only mode:

```text
x1_32rootbench
```

The mode:

```text
1. samples random x in Fp;
2. evaluates the degree-10 y-fiber f_x(y) from Sutherland's FFFc32 equation;
3. computes gcd(f_x(y), y^p - y);
4. records the gcd degree as the number of distinct rational y-roots.
```

This is root-existence/root-counting only, not root extraction.

Build:

```bash
gcc -O3 -Wall -Wextra -o /tmp/pomerance_x132_rootbench_20260602 pomerance.c -lm
```

Compile status:

```text
PASS, no warnings
```

## Comparison To Active Nonsplit X1(16)

Latest active p23 nonsplit workers report:

```text
per-worker active rate ~= 0.108M accepted trials/s
aggregate active rate ~= 1.080M accepted trials/s
```

The direct X1(32) root-existence floor is:

```text
0.001855M random fibers/s
```

So, per single process under the same load environment:

```text
active nonsplit accepted-trial rate / X1(32) root-existence fiber rate
  ~= 0.108 / 0.001855
  ~= 58x
```

Counting only fibers with at least one rational root:

```text
rootful X1(32) fibers/s ~= 1854.779 * 0.6222 ~= 1154/s
```

which is roughly:

```text
108000 / 1154 ~= 94x
```

below one active nonsplit worker's accepted-candidate rate before root
extraction and curve conversion.

Earlier small-prime density calibration suggested direct X1(32) is at best a
constant-factor hazard rung relative to active nonsplit X1(16), not a 50x-100x
hazard gain.

## Decision

Production decision:

```text
Direct X1(32) via generic degree-10 fiber root finding is ruled out as the
next p23 production shard.
```

Reason:

```text
The root-existence floor alone is far too slow. Full sampling would be slower,
and the observed density lift is nowhere near enough to compensate.
```

Research status:

```text
X1(32) remains mathematically validated as the next fixed 2-power torsion rung.
It only revives operationally if a special 2-power parametrization or tower
section avoids generic degree-10 finite-field root finding.
```

Operational conclusion:

```text
Keep the active y-filtered nonsplit X1(16) run.
If it cleanly misses, use the direct-y nonsplit follow-on, not direct X1(32).
```
