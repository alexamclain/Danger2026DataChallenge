# P25 KSY-y Official VPP Submission Archive Contract

Updated: 2026-06-14 21:07 PDT

## Purpose

This contract starts after a concrete p25 `(p,A,x0)` exists.  It separates:

```text
concrete triple
official vpp.py stdout True
complete archive bundle
Lean certificate generated and checked
submission-ready state
```

The p25 moonshot is not finished by an unarchived claim, a failed verifier
run, or a placeholder certificate.

evidence.

## Interpretation

For a future p25 hit, the immediate command is:

```sh
python3 src/vpp.py 10000000000000000000000013 <A> <x0>
```

If and only if it prints `True`, archive the full bundle above and generate
the Lean certificate.  A failed `vpp.py` run kills that concrete payload; a
`True` run with missing logs, hash, environment, provenance, or Lean status is
not a clean finish.
