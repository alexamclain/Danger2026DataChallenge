## Exact Lattice Statement

Use the edge order:

```text
(m1, m2, m4, m8)
```

For an integral vector:

```text
c = (c1, c2, c4, c8)
```

the Hilbert-90 boundary scale is:

```text
(c1 + c2 + c4 + c8) W
```

So the current p25 target boundary requires:

```text
c1 + c2 + c4 + c8 = 1
```

The boundary-zero lattice has rank `3` and is generated by:

```text
m2 - m1 = (-1, 1, 0, 0)
m4 - m1 = (-1, 0, 1, 0)
m8 - m1 = (-1, 0, 0, 1)
```

Every integral `W`-boundary vector decomposes exactly as:

```text
c = m1 + c2*(m2-m1) + c4*(m4-m1) + c8*(m8-m1)
```

Equivalently, every nonunit `W`-boundary vector is:

```text
one unit edge + nonzero boundary-zero lattice content
```

## Minimality

For integral `c` with coefficient sum `1`:

```text
L1(c) >= |sum(c)| = 1
```

Equality `L1(c) = 1` holds only when all nonzero coefficients have the same
sign. Since the coefficients are integral and sum to `1`, equality means
exactly one coefficient is `1` and the other three are `0`.

Thus:

```text
L1-minimal W-boundary vectors = {m1, m2, m4, m8}
```

If a `W`-boundary vector is not a unit edge, then:

```text
L1(c) >= 3
```

because `L1(c) - sum(c)` is twice the total negative mass and is therefore an
even nonnegative integer.
