# Prime-Torsor Obstruction For Abstract Class-Field Towers

This note consolidates the remaining gap in the embedded class-field tower
route.  The point is not that class-field quotients fail to exist.  They exist
in exactly the right degrees.  The point is that, at a prime that splits
completely, an abstract quotient tower gives a torsor of primes above `p`, and
choosing a compatible prime above the ordinary prime of `K` is the same phase
information as choosing an embedded CM coset.

## Setup

Let `O` be a p24 strict CM order, let

```text
G = Cl(O)
```

and let `L_rc/K` be the corresponding ring class field.  The embedded CM roots
over `F_p` are a simply transitive `G`-torsor:

```text
J = {j_g : g in G}.
```

More precisely, the ordinary trace determines a principal prime `frak_p` of
`K` above the rational prime `p`.  This prime splits completely in the ring
class field.  The primes above `frak_p`, not the abstract defining equation
alone, are what reductions modulo `p` choose.

For an intermediate class field `M = L_rc^H`, where `H <= G`, the abstract
quotient has degree

```text
[M:K] = |G/H|.
```

In the third strict p24 target,

```text
G ~= C_205880396014
  = C_(2 * 157 * 211 * 3107441),
H = <g^66254>,
|G/H| = 66254,
|H| = 3107441.
```

The attractive tower is

```text
G > <g^2> > <g^314> > <g^66254>,
relative degrees 2, 157, 211.
```

## Theorem 1: Split Abstract Roots Are Prime Torsors

Assume the ordinary prime `frak_p | p` splits completely in `L_rc`.  Then for
every intermediate field `M=L_rc^H`, the primes of `M` above `frak_p` form a
`G/H`-torsor.  Equivalently, if `alpha` is any primitive abstract generator
for `M`, then the roots of its minimal polynomial modulo `frak_p` are an
unordered `G/H`-torsor in `F_p`.

### Proof

Since `frak_p` splits completely in `L_rc`,

```text
O_{L_rc} tensor_{O_K} F_p ~= product_{g in G} F_p.
```

The Galois action of `G` permutes the factors simply transitively.  Passing to
`H`-invariants gives

```text
O_M tensor_{O_K} F_p ~= (O_{L_rc} tensor_{O_K} F_p)^H
                     ~= product_{G/H} F_p.
```

The primitive idempotents, equivalently the primes of `M` above `frak_p`, are
therefore indexed by `G/H`, and `G/H` acts simply transitively on them.

The same statement holds at each layer of a tower.  If `H_child <= H_parent <=
G` are subgroups in class-group notation, the child primes above one parent
prime are an `H_parent/H_child`-torsor.

## Theorem 2: Abstract Towers Do Not Pair Themselves With `j`

Let `B_H` be the reduced abstract quotient algebra

```text
B_H = O_{L_rc^H} tensor_{O_K} F_p ~= F_p^(G/H).
```

Let `A_H` be the split embedded period algebra generated by an actual
`H`-coset invariant of the CM roots, for example

```text
y_{gH} = sum_{h in H} j_{gh},
A_H ~= product_{gH in G/H} F_p.
```

Both algebras are split finite etale `F_p`-algebras of degree `|G/H|`, and
both carry transitive `G/H` actions.  But an isomorphism

```text
B_H -> A_H
```

is extra data.  The set of `G`-compatible isomorphisms is itself a `G/H`
torsor.  Thus an abstract class-field polynomial plus the embedded CM period
set leaves exactly `|G/H|` possible pairings, unless a relation to `j` or a
phase convention is supplied.

### Proof

Choose one primitive idempotent of `B_H` and one primitive idempotent of
`A_H`.  There is then a unique `G/H`-equivariant isomorphism carrying the
first to the second.  Changing the target idempotent by an element of `G/H`
changes the isomorphism, and these are all the compatible isomorphisms.  Hence
the isomorphism set is a torsor under `G/H`; it has no distinguished element
from the abstract algebraic data alone.

In computational terms, reducing a `bnrclassfield` polynomial modulo `p`
gives split roots, but it does not say which abstract root corresponds to
which embedded cycle sum, nor which recovery polynomial in `j` lies above it.

The tiny check

```text
p24/abstract_vs_embedded_quotient_toy.py
```

shows this in degree `2`: the abstract roots of `x^2+3` and the embedded
cycle-sum roots both split over `F_103`, but the class-field equation alone
does not choose one of the two pairings.

The non-genus check

```text
p24/abstract_embedded_pairing_non_genus_toy.py
```

is more diagnostic.  For `D=-2239`, `h=35`, quotient degree `5`, both the
abstract quotient and the embedded split-cycle period quotient split over
`F_2243`, but there is no affine or Mobius set map from the abstract roots to
the embedded period roots.  This does not prove a universal lower bound, but
it shows that the pairing behaves like genuine extra embedded data even in a
small non-genus case.

## Theorem 3: No Seedless Relative Child Section

For a nontrivial tower layer

```text
H_child < H_parent <= G,
```

there is no `G`-equivariant section

```text
G/H_parent -> G/H_child.
```

Equivalently, once a parent quotient root is known, a seedless natural rule
cannot name one child root above it.  The child set is an
`H_parent/H_child`-torsor.

### Proof

If `s(H_parent)=aH_child`, then for every `k in H_parent`,

```text
s(H_parent) = s(kH_parent) = k s(H_parent) = kaH_child.
```

Thus `a^{-1}ka in H_child`.  The p24 class groups here are abelian, so
`k in H_child`.  Hence `H_parent <= H_child`, contradicting
`H_child < H_parent`.

This is the same obstruction isolated in

```text
p24/tower_section_obstruction.md
```

but phrased in prime-above-`p` language.

## Consequence For Explicit Class-Field Machinery

Explicit class-field tools can do at least one of the following:

```text
1. define abstract quotient fields;
2. prove the desired splitting at p;
3. generate ray class fields and list conjugates by Shimura reciprocity;
4. compute all subgroup traces or norms after enumerating the class action.
```

None of these alone supplies the missing embedded map

```text
abstract quotient root -> embedded H-period -> recovery polynomial in j.
```

To become a certificate algorithm, a proposed tower must provide an embedded
generator or relation such as

```text
F_1(Y_1) = 0,
F_2(Y_2, Y_1) = 0,
F_3(Y_3, Y_2) = 0,
R(J, Y_3) = 0,
```

where each `F_i` is not merely an abstract defining equation but is paired to
the CM `j`-torsor modulo `p`.  Arbitrary finite-field root choice is fine once
such embedded relations are known; it is not a substitute for the relations.

## Relation To The Degree And Support Bounds

The prime-torsor obstruction is complementary to the finite-field degree
theorem:

```text
p24/finite_field_selector_degree_theorem.md
```

If a scalar coordinate is actually constant on `H`-cosets, one fiber contains
`|H|` distinct CM points, so its degree is at least `|H|`.  For the third p24
target this lower bound is

```text
3107441,
```

which is below `sqrt(p)`.  Therefore a deliberate growing-degree recovery
object is not ruled out.  What is ruled out is the idea that an abstract tower
or bounded local coordinate can provide the same embedded quotient for free.

For additive oriented Hecke formulas, the support lemma gives the parallel
statement:

```text
p24/structural_lifting_support_lemma.md
```

Once a formula restricts to a faithful group-algebra operator on the CM
torsor, producing an `H`-period forces the exact projector

```text
e_H = sum_{h in H} [h],
```

so the support is at least `|H|`.  Bounded p-specific vanishing identities are
also constrained by

```text
p24/finite_field_modular_zero_lemma.md
```

through pole degree and the Tate-cusp nonidentity test.

## What Remains Open

This theorem does not close the surviving positive route.  It identifies its
required input:

```text
compute embedded non-genus relative class-character traces for the odd
157 and 211 tower layers, paired back to j, without enumerating Cl(O)
or constructing the full class polynomial.
```

The formal trace identity is

```text
T_s(aK) = sum_{k in K} chi_s(k) j_{ak}.
```

Once these traces are known, inverse Fourier transform gives the relative
child periods, and the final recovery polynomial has degree `3107441`.

For the third p24 trace, genus data supplies only the top degree-2 layer:

```text
D_K = -599 * 1089874116562502921057,
G/G^2 has order 2.
```

The odd `157` and `211` phases live inside the principal genus and are
invisible to genus characters, quadratic subfields, Kronecker symbols, and
Redei refinements.

So the class-field tower route is now sharply split:

```text
abstract tower: available, but unpaired with j;
bounded local identities: degree/support-obstructed;
embedded odd relative periods: still the only live theorem target.
```
