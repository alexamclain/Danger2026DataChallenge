# Relative Resolvent Content Certificate

This note turns the harmful-dual-coset condition into an exact finite-field
certificate target.

## Algebraic Closure Statement

Let `h=m n`, `H=<m>`, and define

```text
P_u(a) = sum_{k=0}^{n-1} zeta_n^(a k) j_{u+m k},
          0 <= u < m.
```

The lemma in `harmful_dual_coset_relative_resolvent_lemma.md` says that the
dual coset `a+Q_H` vanishes if and only if all `P_u(a)` vanish.

## Packetized `F_p` Statement

Over `F_p`, a nontrivial `H`-character `a` must be treated together with its
Frobenius orbit

```text
O(a) = {a, p a, p^2 a, ...} in (Z/nZ)^*.
```

Let `f_a(X)` be the minimal polynomial of `zeta_n^a` over `F_p`, and form the
fiber polynomials

```text
J_u(X) = sum_{k=0}^{n-1} j_{u+m k} X^k.
```

Then the whole Frobenius packet of harmful dual cosets vanishes if and only if

```text
J_u(X) == 0 mod f_a(X)      for every 0 <= u < m.
```

Equivalently, the vector

```text
(J_0 mod f_a, J_1 mod f_a, ..., J_{m-1} mod f_a)
```

is the zero vector in the degree-`deg(f_a)` field `F_p[X]/(f_a)`.

## Exact Certificate

The exact certificate for one orbit is a content/Bezout certificate:

```text
there are polynomials B_u(X) such that
sum_u B_u(X) J_u(X) == 1 mod f_a(X).
```

This proves that the relative-resolvent vector is not identically zero modulo
the selected prime.  In characteristic-zero language, it proves that the ideal
generated by the relative resolvents is not contained in the selected prime
above `p`.

For the third p24 target there are only eight nontrivial `H`-character
Frobenius orbits:

```text
n = 3107441,
ord_n(p) = 388430,
(n-1)/ord_n(p) = 8.
```

So a complete relative-resolvent content certificate would consist of eight
such nonzero-vector certificates.

## Product/Norm Variant

A product certificate is simpler but stronger than needed.  If

```text
prod_u (J_u mod f_a) != 0
```

in `F_p[X]/(f_a)`, then no harmful packet vanishes.  However the product is
zero as soon as one fiber is zero, while the harmful event requires every
fiber to be zero.  Thus product nonvanishing is sufficient, not equivalent.

The ideal-content/Bezout formulation above is the exact target.

There is also a scalar sufficient certificate:

```text
E_a = sum_u P_u(a) P_u(-a).
```

If `E_a != 0`, then the harmful packet cannot vanish.  This is weaker than a
content certificate and can suffer cancellation, but it reduces one orbit to a
single toric autocorrelation/nonvanishing statement.  See:

```text
p24/relative_energy_certificate.md
```

The conjugacy/norm interpretation is recorded in:

```text
p24/relative_resolvent_conjugacy_boundary.md
p24/relative_resolvent_split_algebra_theorem.md
```

It explains why a quotient product is a clean sufficient certificate but not
an equivalent one, and why the exact condition is the content ideal generated
by the quotient conjugates.

## Embedded Phase Barrier

In the characteristic-zero class field, the `P_u(a)` are conjugate up to roots
of unity under the quotient class action.  After choosing one prime over `p`,
those conjugates land at conjugate primes, not automatically at the same
finite-field embedding.  Therefore a nonvanishing statement for one conjugate
does not by itself prove the selected embedded vector is nonzero on every
fiber.

This is the same embedded phase barrier that blocked the abstract class-field
tower route: the abstract quotient roots are easy to name, but pairing them
with the selected finite-field `j` periods is the hard part.

## Probability Scale

For one p24 orbit, the relative vector has length

```text
m = 66254
```

over a field of size

```text
p^388430.
```

A random-vector model predicts the all-zero event with probability roughly

```text
p^(-66254*388430).
```

This explains why the small scans see no natural harmful packet, but it is
not a proof for the selected prime.  The proof still needs an explicit
content, norm, modular-zero, or p-adic nonvanishing argument.

## Current Bottom Line

The missing theorem can now be stated as:

```text
For each of the eight nontrivial Frobenius orbits of H-characters in the
third p24 target, the relative-resolvent content vector is nonzero modulo the
selected prime above p.
```

This is narrower than full reduced normality and exact enough to guide either
a new class-field identity or a finite-field certificate.
