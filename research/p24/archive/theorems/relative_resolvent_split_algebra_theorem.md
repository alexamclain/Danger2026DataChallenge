# Relative Resolvent Split-Algebra Theorem

This note sharpens the harmful-dual-coset condition from a vector statement to
an ideal statement in the split reduction of the class field.

## Setup

Let

```text
h = m n,              G = <sigma>,              H = <sigma^m>,
```

and write the embedded CM roots as

```text
j_i = sigma^i(j_0),          0 <= i < h.
```

After adjoining `mu_n`, define the relative `H`-character resolvent

```text
Theta_a = sum_{k=0}^{n-1} zeta_n^(a k) sigma^(m k)(j_0),
          a not == 0 mod n.
```

For `0 <= u < m`, the fiber sums are

```text
P_u(a) = sum_{k=0}^{n-1} zeta_n^(a k) j_{u+m k}
       = sigma^u(Theta_a).
```

The `H`-eigen relation is

```text
sigma^m(Theta_a) = zeta_n^(-a) Theta_a.
```

## Split-Algebra Form

Fix a prime `P` above the selected ordinary prime over `p`, together with a
chosen reduction of `zeta_n`.  Since the ordinary prime splits completely in
the ring class field, reduction of the class-field algebra gives a product of
split factors indexed by `G`.

In this split algebra, the component of `Theta_a` at the translate
`sigma^(u+m l) P` is

```text
sigma^(u+m l)(Theta_a)
  = zeta_n^(-a l) P_u(a).
```

Because the root of unity is a unit, the number of zero split components is

```text
n * #{u : P_u(a) == 0 mod P}.
```

Consequently:

```text
one fiber P_u(a) vanishes
  iff Theta_a vanishes on the n split components above that quotient fiber;

all fibers P_u(a) vanish
  iff Theta_a is zero in the entire split class algebra modulo the chosen
     cyclotomic prime over p.
```

Equivalently, the harmful dual-coset condition is

```text
Theta_a in p_E O_{L E},
```

where `E` is the root-of-unity field and `p_E` is the selected prime of `E`
below `P`.  In norm language, one zero fiber forces an `n`-fold split-prime
divisibility of `Norm_{L E/E}(Theta_a)`, while harmful all-fiber vanishing
forces the full `h`-fold split-prime divisibility.

## Relation To Earlier Certificates

This theorem explains the hierarchy of certificates.

```text
exact harmful condition:
  Theta_a is zero in the split reduced class algebra;

exact finite-field packet condition:
  J_u(X) == 0 mod f_a(X) for every u;

content certificate:
  the ideal generated by the J_u mod f_a is the unit ideal;

strong product certificate:
  every P_u(a), or equivalently every split quotient block, is nonzero.
```

The split-algebra condition is stronger than a single norm vanishing and more
precise than the product shortcut: it records how many split components vanish.

## p24 Consequence

For the third p24 target,

```text

ord_n(p) = 388430.
```

A harmful Frobenius packet is therefore not a one-character accident.  For one
algebraic relative character it would make `Theta_a` zero on all
`205880396014` split class components over the selected cyclotomic prime; over
`F_p` the event propagates across a Frobenius orbit of `388430` nontrivial
relative characters.

This is a cleaner p-adic target:

```text
prove Theta_a not in p_E O_{L E}
for one representative of each of the eight nontrivial H-character
Frobenius orbits.
```

But the crude height route still does not close it.  The principal singular
modulus has logarithmic size about

```text
pi * sqrt(|D_K|) ~= 2.54e12,
```

whereas the per-quotient divisibility threshold is only

```text
m * log(p) ~= 3.64e6.
```

So `p^h` divisibility of the relative norm is not contradicted by elementary
archimedean size bounds.  A proof still needs a structural p-adic unit
argument, a phase-aware class-character trace formula, or an explicit finite
field content certificate.
