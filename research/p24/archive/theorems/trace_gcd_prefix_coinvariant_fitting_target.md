# Trace-GCD Prefix Coinvariant Fitting Target

Date: 2026-06-06

## Point

The additive Hilbert-90 quotient formulation turns the p24 four-prefix
problem into a rectangular p-local Fitting statement.

The quotient map is:

```text
bar_phi_B : R^4 -> E/(tau_R - 1)E,
(y_j) -> [sum_{j in {2,3,5,6}} y_j*S_j].
```

As an `F_p`-linear map it has:

```text
source rank = 4*35 = 140,
target rank = 156.
```

Therefore the p-unit object is not a single square determinant.  It is the
maximal-minor ideal of this `156 x 140` map, equivalently the appropriate
Fitting ideal of its cokernel.

## P-Local Statement

Let `O_p` be the selected p-local coefficient ring, with p-integral lattices
for `R`, `E`, and the coinvariant quotient

```text
Q = E/(tau_R - 1)E.
```

Let

```text
Phi_B : O_R^4 -> Q
```

be the p-integral lift of `bar_phi_B`.

The prefix theorem is:

```text
Fitt_16(coker Phi_B) = O_p.
```

Equivalently, the ideal generated by all `140 x 140` minors of any p-integral
matrix for `Phi_B` is the unit ideal.  After reduction mod p this says:

```text
rank(Phi_B mod p) = 140.
```

This is exactly the prefix rank input:

```text
rank_Fp(A_B) = 140,
dim_Fp ker(A_B : L -> R^4) = 16.
```

The index `16` in `Fitt_16` is the expected rank of the cokernel:

```text
156 - 140 = 16.
```

## Relation To Previous Forms

The trace map gives an isomorphism:

```text
E/(tau_R - 1)E  ~=  L
```

up to multiplication by the p-unit `[E:L]=35`.  Under this isomorphism,
`Phi_B` becomes the trace-adjoint map:

```text
R^4 -> L,
(y_j) -> Tr_{E/L}(sum_j y_j*S_j).
```

So the following are the same p-local theorem:

```text
1. Phi_B has unit maximal-minor ideal;
2. bar_phi_B is injective mod p;
3. the trace-adjoint A_B^* is injective;
4. the four prefix trace blocks have rank 140;
5. some 140 x 140 prefix Plucker pivot is a p-unit.
```

This note names the determinant-line object for item 1.  The actual proof
still has to construct `Phi_B` from the mixed CM resolvent pairings:

```text
S_j = H_{157,211}(1,v_j) = <A_1,B_{v_j}>
```

and prove the p-local Fitting ideal is the unit ideal without enumerating the
class set.

## Normal-Basis Coefficient Form

Choose any normal basis `alpha_i` of `R/F_p`, with trace-dual basis
`alpha_i^vee`.  Then every `S in E` has the expansion

```text
S = sum_i Tr_{E/L}(alpha_i*S) * alpha_i^vee.
```

Consequently the same prefix Fitting theorem is the rank statement

```text
rank_Fp{Tr_{E/L}(alpha_i*S_j) :
        j in {2,3,5,6}, 0 <= i < 35} = 140.
```

## Why This Is Useful

The residual Schur-complement route asked for a named prefix Plucker pivot.
That is coordinate-dependent.  The coinvariant Fitting target is intrinsic:
it says the whole maximal-minor ideal is a unit.  Once this is proved, any
p-unit pivot can be chosen for the Schur representative, and the remaining
`16 x 16` quotient-tail determinant is the existing local-unit target.

Thus the representative `140+16` proof can be split as:

```text
prefix:
  Fitt_16(coker Phi_B) = O_p;

tail:
  det(first16(O1) on ker A_B) in O_p^*.
```

The second line is the linearized-resultant p-unit already isolated in:

```text
p24/linearized_trace_gcd_certificate_boundary.md
p24/trace_gcd_two_linearized_resultant_target.md
```
