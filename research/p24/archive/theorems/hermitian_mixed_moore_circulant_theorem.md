# Hermitian Mixed Moore-Circulant Theorem Candidate

This note records the CS/coding-theory import that survived the latest
Hermitian Schur-boundary tests.

## Mixed DFT Block

For the Hermitian packet kernel

```text
K(r,s)=Tr_packet(F_r(X)F_s(X^-1)),
```

the mixed `c x d` character-pairing block is

```text
H_{c,d}(u,v)
  = sum_{r,s} zeta_c^(u*r) zeta_d^(v*s) K(r,s),
    1 <= u < c, 1 <= v < d.
```

Since `K(r,s)` lies in the base field, Frobenius gives the exact identity

```text
H(q^a*u0, q^b*v0) = H(u0, q^(b-a)*v0)^(q^a).
```

After ordering rows and columns by `q`-orbits, every orbit-pair block is
therefore a Moore/Frobenius-circulant matrix generated by one seed row.

## Skew-Linearized Kernel Form

Full row rank of the mixed block is equivalent to the following
interleaved-Gabidulin style condition.

There is no nonzero vector `x=(x_0,...,x_155)` in the scalar-extension field
such that for every column orbit `j` and every `b mod 35`,

```text
sum_{a=0}^{155} x_a * b_j(b-a mod 35)^(p^a) = 0.
```

Equivalently, no nonzero skew-linearized convolution operator of
`p`-degree `<156` annihilates all six seed cycles.  This is a concrete
rank-metric/PIT theorem surface for the Schur correction.

## Lang-Normality Refinement

The skew-cyclic condition can be trivialized further by Lang/Hilbert-90 style
linear algebra.  This is recorded in:

```text

p24/hermitian_mixed_lang_normality_theorem.md
```

For a right orbit of length `R`, the semilinear shift

```text
T(s)_b = sigma(s_{b-1})
```

has fixed-vector basis

```text
u_alpha(b)=sigma^b(alpha),       alpha in F_{q^R}.
```

After changing coordinates by this Moore matrix, a mixed row orbit becomes an
ordinary Moore matrix.  Therefore full row-orbit rank is equivalent to an
`F_q`-normality condition on the transformed seed coordinates.

For p24, the target becomes:

```text
the 210 Lang-trivialized mixed Hermitian seed coordinates have
F_p-span dimension at least 156.
```

This is currently the sharpest finite-field theorem statement for the mixed
Schur correction.
