# Hermitian Double-Marginal Fourier Form

This note diagonalizes the centered CRT double-marginal blocks from
`hermitian_double_marginal_formula.md`.

## Statement

For the Hermitian packet kernel

```text
K(r,s)=Tr_packet(F_r(X)F_s(X^-1)),
```

and components `c,d | m`, define

```text
M_{c,d}(a,b)=sum_{r==a mod c, s==b mod d} K(r,s).
```

The trace-zero block `U_c x U_d` is the centered table of `M_{c,d}`.  After
adjoining the relevant roots of unity, its rank is the rank of the nonzero
double DFT:

```text
H_{c,d}(u,v)
  = sum_{a mod c, b mod d} zeta_c^(u*a) zeta_d^(v*b) M_{c,d}(a,b),
    1 <= u < c, 1 <= v < d.
```

Equivalently,

```text
H_{c,d}(u,v)
  = sum_{r,s} zeta_c^(u*r) zeta_d^(v*s) K(r,s).
```

This is the mixed K-character pairing matrix for the `c` and `d` CRT
component frequencies.  The centered marginal matrix and the nonzero DFT
matrix differ by invertible Fourier transforms on the trace-zero hyperplanes.

## p24 Consequence

For p24 the Schur correction can be described in either equivalent form:

```text
centered CRT double marginals of K(r,s),
or mixed nonzero K-character pairing matrices.
```

The large mixed block is:

```text
157 x 211 centered marginal
<=> 156 x 210 nonzero-frequency mixed pairing matrix.
```

This aligns the Hermitian route with the earlier tensor/rank-condenser
language:

```text
component p-units = diagonal character-pairing blocks;
Schur correction = mixed character-pairing determinant;
packet norm = product over eight H-character packets.
```

## Current Theorem Target

The missing arithmetic theorem can now be stated as:

```text
The mixed nonzero K-character Hermitian pairing matrix for the p24
CRT-axis frequencies is nonsingular modulo the selected prime,
packet by packet, after accounting for the diagonal component blocks.
```

This is still a coupled selected-prime p-unit theorem.  The value of the
Fourier form is that it replaces a large centered residue table by the
character-pairing matrix one expects from class-field theory.

## Frobenius-Orbit Refinement

The mixed character-pairing block has an additional semilinear structure
coming from the fact that the kernel entries lie in the base field:

```text
H(q^a*u0, q^b*v0) = H(u0, q^(b-a)*v0)^(q^a).
```

I added:

```text
p24/hermitian_mixed_frobenius_orbit_audit.py
p24/hermitian_mixed_moore_circulant_theorem.md
```

For the p24 `157 x 211` mixed block this gives:

```text
157 side: one orbit of length 156
211 side: six orbits of length 35
```

so the `156 x 210` matrix is a six-block Moore/Frobenius-circulant object
generated by six length-35 seed cycles.  The corresponding theorem target is
that no nonzero skew-linearized convolution operator of `p`-degree `<156`
annihilates all six seed cycles.

identity with zero failures and found every orbit block full possible rank in
the tested cases:


This imports the useful part of rank-metric/Gabidulin theory: not an
off-the-shelf determinant proof, but a smaller skew-PIT/class-field p-unit
surface for the mixed Schur correction.
