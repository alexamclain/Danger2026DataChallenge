# Trace-GCD Prefix Normal-Basis Coefficients

Date: 2026-06-06

## Point

The coinvariant prefix theorem can be written without an abstract quotient.
Choose a normal basis

```text
alpha_0,...,alpha_34
```

of `R/F_p`, and let

```text
alpha_0^vee,...,alpha_34^vee
```

be its trace-dual basis:

```text
Tr_{R/F_p}(alpha_i * alpha_j^vee) = delta_ij.
```

For any `S in E=L R`, the relative trace coefficients

```text
c_i(S) = Tr_{E/L}(alpha_i * S) in L
```

recover the `R`-coordinate expansion

```text
S = sum_i c_i(S) * alpha_i^vee.
```

Therefore the quotient/adjoint prefix map

```text
R^4 -> E/(tau_R - 1)E ~= L,
(y_j) -> [sum_j y_j*S_j]
```

is injective if and only if the `140` elements

```text
c_i(S_j) = Tr_{E/L}(alpha_i * S_j),
j in {2,3,5,6}, 0 <= i < 35,
```

are `F_p`-linearly independent in `L`.

This is the same p-local Fitting theorem:

```text
Fitt_16(coker(R^4 -> E/(tau_R-1)E)) = O_p.
```

The Fitting ideal is generated by the `140 x 140` minors of the coordinate
matrix whose rows are the `c_i(S_j)` in any p-integral basis of `L`.

## Why It Helps

The Hilbert-90 statement says no nonzero four-prefix combination is a
right-cyclotomic coboundary:

```text
sum_j y_j*S_j = tau_R(Z)-Z
  =>
y_j=0.
```

The normal-basis statement says the same thing as a concrete coefficient
independence theorem in the left cyclotomic field:

```text
{Tr_{E/L}(alpha_i*H_{157,211}(1,v_j))}_{i,j}
```

has rank `140` over `F_p`.

This is a better bridge to explicit class-field period formulas.  Using the
mixed resolvent pairing

```text
S_j = H_{157,211}(1,v_j) = <A_1,B_{v_j}>,
```

the missing theorem becomes:

```text
the 140 right-normal-basis Fourier coefficients of the four selected mixed
Hermitian pairings form an F_p-independent family in F_p(mu_157).
```

Because `[R:F_p]=35` is odd and `p` is odd, `R/F_p` also admits a self-dual
normal basis.  In such a basis `alpha_i^vee=alpha_i`, so the formula has no
Gram-matrix correction.  The trace-dual version above is basis-independent
and avoids making the self-dual choice part of the certificate.

For p24 there is an even more explicit choice: a type-6 Gaussian normal basis
coming from

```text
211 = 6*35 + 1,    ord_211(p)=35.
```

This is recorded in:

```text
p24/trace_gcd_prefix_gaussian_period_basis.md
p24/trace_gcd_prefix_gaussian_normal_basis_toy.py
```

It writes the right basis as six-term periods

```text
eta_i = sum_{c in C0} zeta_211^(p^i c),
```

so the prefix determinant becomes a phase-aware cyclic convolution of the
right Frobenius orbit of the four mixed periods.

The right-cycle DFT of that convolution uses `F_p(mu_35)` coefficients.  The
rank-safe version lives in `L tensor F_p(mu_35)`, not in a collapsed copy of
`L`; see:

```text
p24/trace_gcd_prefix_gaussian_dft_scalar_extension_boundary.md
```

## Proof Obligation

The remaining arithmetic theorem is now:

```text
rank_Fp(
  Tr_{E/L}(alpha_i * H_{157,211}(1,v_j))
    for j in {2,3,5,6}, 0 <= i < 35
) = 140.
```

Equivalently, the Moore/Chow section of these `140` left-field coefficients
is a p-unit:

```text
Delta_140(c_i(S_j)) in O_p^*.
```

This is exactly the prefix factor in the `140+16` residual Moore/Chow split.
It does not prove the quotient-tail p-unit, but it supplies the named prefix
section that the quotient-tail Schur complement needs.

```text

  p24/trace_gcd_prefix_normal_basis_toy.py
```

The toy verifies in coprime finite extensions that:

```text

```

It includes forced zero and repeated-period controls, plus random independent
