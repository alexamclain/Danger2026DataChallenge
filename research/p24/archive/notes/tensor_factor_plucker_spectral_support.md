# Tensor Factor Plucker Spectral Support

This note records the formal support bound behind the beta-product recurrence
route.

## Lemma

Let `B/E` be one p24 tensor factor, generated by `theta`, and let

```text
O = <p^ord_m(p)> mod n
```

be the E-Frobenius orbit of the selected `n`-th root coordinate.  For fixed
`E`-linear functionals `L_j : B -> E`, each beta-shifted entry

```text
beta |-> L_j(theta^(-beta) x)
```

has beta-character support contained in `O`.

Therefore any `d x d` Plucker coordinate

```text
P(beta) = det( L_j(theta^(-beta) x_i) )_{1 <= i,j <= d}
```

has beta-character support contained in the `d`-fold additive sumset:

```text
dO = O + ... + O  mod n.
```

Over `E`, the nonzero characters of order `n` split into Frobenius orbit
factors indexed by multiplicative `O`-cosets.  Since `dO` is invariant under
multiplication by `O`, the recurrence degree of `P(beta)` is bounded by:

```text
zero_character_used
  + 5549 * (# of nonzero O-cosets met by dO).
```

This is only an upper bound.  A special CM Plucker coordinate can have smaller
support by cancellation, as in the small `D=-10919` projected row where the
degree-13 sequence has recurrence order `7`.

## p24 Consequence

For p24:

```text
|O| = 5549,
number of nonzero O-cosets = 560.
```

The exact support audit reports:

```text
2O: 557 nonzero O-cosets, zero character absent;
3O: 560 nonzero O-cosets, zero character present.
```

Thus any Plucker coordinate involving three independent beta-shifted entry
characters has full support available.  The actual p24 exterior products have
ranks:

```text
Omega_1:    158,
Omega_211:  210,
Omega_3:    368.
```

So recurrence/resultant compression is not forced by representation support.
To beat beta enumeration, one must prove a **CM-specific support collapse** or
prove a p-unit theorem for the full-support resultant.

The trace-coordinate support audit:

```text
p24/tensor_factor_trace_coordinate_support_audit.py
p24/tensor_factor_trace_coordinate_support.md
```

shows the same boundary for ordinary coordinate minors: six trace-coordinate
cosets already fill all of `Z/nZ` in every tested random six-coordinate
sample, while p24 coordinate minors select `158`, `210`, or `368` coordinates.

This is the precise CS/PIT boundary: the resultant surface is valid, but a
rank-condenser/superregularity theorem still has to use arithmetic structure,
not just sparse spectral support.
