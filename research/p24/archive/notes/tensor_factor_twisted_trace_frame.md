# Tensor Factor Twisted Trace Frame

This is the current sharpest refinement of the one-factor Moore target.

## Setup

For one p24 tensor factor:

```text
B/E,          [B:E] = 5549 = 31 * 179
E = F_p(mu_m)
Q = |E| = p^5460
C = F_{Q^179} inside B
[B:C] = 31
```

Let `theta` be the image of the packet coordinate in the selected factor.  For
`k >= 1`, define the twisted trace frame:

```text
T_k : B -> C^k
T_k(x) = (
  Tr_{B/C}(x),
  Tr_{B/C}(theta*x),
  ...,
  Tr_{B/C}(theta^(k-1)*x)
).
```

This is an `E`-linear map.  Therefore:

```text
rank_E { T_k(R_s) : s in S_axis } = 368
  => rank_E { R_s : s in S_axis } = 368.
```

The formal implication is Lean-checked in:

```text
p24/lean/TraceFrameGate.lean
```

combined with:

```text

p24/lean/ScalarExtensionGate.lean
```

## Why This Beats The Plain Subfield Split

Plain traces to proper subfields have total target dimension:

```text
31 + 179 = 210 < 368.
```

So they cannot certify the full p24 axis rank.

The degree-179 twisted trace frame with `k=3` has target dimension:

```text
3 * 179 = 537 > 368.
```

This is still much smaller than the original one-factor ambient dimension:

```text
5549.
```

Thus the finite certificate surface can move from a degree-5549 Moore
determinant to a rank statement in three degree-179 trace coordinates.

## p24 Theorem Target

For every one of the eight p24 H-character packets and one tensor factor
`B/E`, prove:

```text
rank_E {
  (
    Tr_{B/C}(R_s),
    Tr_{B/C}(theta R_s),
    Tr_{B/C}(theta^2 R_s)
  )
  : s in S_axis
} = 368,
```

where `C=F_{Q^179}`.

Equivalently, prove nonvanishing of one `368 x 368` coordinate minor of this
trace-frame matrix after choosing an `E`-basis of `C^3`.

The intrinsic version is the exterior product:

```text
∧_{s in S_axis} T_3(R_s) != 0
```

inside:

```text
Exterior_E^368(C^3).
```

Coordinate minors are just Plucker coordinates of this exterior vector.

The dual-basis refinement names a canonical coordinate map:

```text
x |-> Top_3(g'(theta) * x),
```

where `Top_3` extracts the `theta^30`, `theta^29`, and `theta^28`
coefficients over `C`.  The live p24 theorem is that this map is injective on
the selected 368-dimensional axis span.
