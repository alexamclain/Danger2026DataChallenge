# Hermitian Axis Lattice Primitivity

This note restates the current p24 determinant target as a local lattice
problem.

## Local Lattice Statement

For each p24 H-packet factor

```text
A_a = F_p[X]/(f_a),       deg(f_a)=388430,
```

let

```text
Y_0, ..., Y_367 in A_a
```

be the images of the standard basis of `W_axis`.  The Hermitian pairing is

```text
<x,y> = Tr_{A_a/F_p}(x * y^(p^(d/2))).
```

The certificate target is:

```text
det(<Y_i,Y_j>) != 0 mod p
```

for each of the eight packet factors.

Equivalently:

```text
span_Fp{Y_i} ∩ span_Fp{Y_i}^perp = 0.
```

In number-field language, before reduction at a prime `P | p`, let `Lambda_a`
be the `Z_P`-lattice generated by the same axis elements inside the selected
packet algebra.  The stronger p-adic theorem is:

```text
disc_Hermitian(Lambda_a) is a P-unit.
```

This is a primitive local-lattice theorem.  It is stronger than saying the
axis images are independent mod `P`, and weaker/different from full
`K`-normality.

## Why Local Normal Bases Are Not Enough

Since the p24 prime is ordinary and unramified in the CM class field, the full
local algebra has a perfect trace pairing and admits normal bases.  But this
is existential.  It says some lattice basis is unimodular, not that the
selected level-1 `j` axis lattice is.

I added the finite-field warning:

```text

```

```text

```

with middle Frobenius involution `x -> x^(3^2)`.  The nonzero vector `a^3`
has

```text
Tr(a^3 * (a^3)^(3^2)) = 0.
```

Thus even an unramified even-degree packet can have Hermitian trace-isotropic
lines.  The p24 proof must use the arithmetic of the CM axis lattice itself.

## Origin Invariance

The Hermitian determinant is better than merely empirically stable across
origins.  The finite identity is recorded in:

```text
p24/hermitian_origin_invariance_theorem.md
```

If an origin shift `s` has CRT coordinates

```text
s == n*alpha + m*beta mod h,
```

then

```text
F'_r(X) = X^(-beta) F_{r+alpha}(X).
```

The axis space is stable under `r -> r+alpha`, and the Hermitian pairing
cancels the monomial because `p^(d/2) == -1 mod n`:

```text
<X^(-beta)x, X^(-beta)y> = <x,y>.
```

Thus origin shift gives a unimodular change of the axis basis and leaves the
Hermitian determinant's p-unit status unchanged.  The p24 theorem is still
selected-prime and packetwise, but not selected-origin in any extra sense.
