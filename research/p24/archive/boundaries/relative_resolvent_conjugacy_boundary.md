# Relative Resolvent Conjugacy Boundary

This note records what the class action gives for the relative-resolvent
certificate target, and what it does not give.

## Class-Field Conjugacy

Let the cyclic class group be generated by `sigma`, with

```text
h = m n,
H = <sigma^m>.
```

Over the ring class field with `mu_n` adjoined, extend `sigma` to fix
`zeta_n`, and define

```text
Theta_a = sum_{k=0}^{n-1} zeta_n^(a k) sigma^(m k)(j_0).
```

Then the fiberwise relative sums are just quotient conjugates:

```text
P_u(a) = sigma^u(Theta_a).
```

Also

```text
sigma^m(Theta_a) = zeta_n^(-a) Theta_a,
```

so membership of `Theta_a` in a prime above `p` is invariant along the
`H`-orbit of that prime.

## Selected-Prime Form

Fix a prime `frak P` of the class-field/root-of-unity compositum above the
selected finite-field embedding.  The harmful algebraic coset condition

```text
P_u(a) == 0 mod frak P      for every 0 <= u < m
```

is equivalent to

```text
Theta_a in intersection_u sigma^(-u)(frak P).
```

Because of the `H`-eigen relation, this is equivalently membership in every
class-group translate of `frak P` inside the fixed cyclotomic packet.

This is the embedded prime-pairing barrier in ideal language: the abstract
class-field conjugates are easy to name, but the selected finite-field
embedding chooses one prime above `p`, and the certificate needs information
about the whole quotient orbit of that prime.

## Norm And Product

The quotient product

```text
N_a = product_{u=0}^{m-1} P_u(a)
```

is a natural relative norm.  If

```text
N_a not == 0 mod frak P,
```

then no fiber vanishes, so the harmful packet is ruled out.

But the converse is false: `N_a == 0 mod frak P` only says at least one
quotient fiber vanishes, while the harmful event requires every fiber to
vanish.  Thus product/norm nonvanishing is a strong sufficient certificate,
not the exact certificate.

The exact condition is the content ideal

```text
C_a = (P_0(a), P_1(a), ..., P_{m-1}(a)).
```

The harmful packet vanishes if and only if

```text
C_a <= frak P.
```

Equivalently, one wants a Bezout/content certificate showing `1 in C_a` after
reducing modulo the Frobenius minimal polynomial `f_a`; this is the finite
field statement in `relative_resolvent_content_certificate.md`.

The split-algebra refinement is recorded in:

```text
p24/relative_resolvent_split_algebra_theorem.md
```

It says that if exactly `b` quotient fibers vanish, then `Theta_a` is zero on
exactly `b*n` split class-field components.  Thus the harmful condition is
equivalent to `Theta_a` being zero in the entire split class algebra modulo
the selected cyclotomic prime.  In norm language, one zero fiber gives an
`n`-fold split-prime divisibility, while harmful all-fiber vanishing gives the
full `h`-fold divisibility.

## Zero-Lemma Conditional

There is a tempting finite-field modular-zero argument:

1. Realize `P_u(a)` as a rational function `F_a` on a modular or
   correspondence cover.
2. Harmful vanishing makes `F_a` vanish on all class-group translates of the
   selected ordinary CM point.
3. If the pole degree of `F_a` is below that CM orbit size and the cusp
   expansion is not identically zero, `finite_field_modular_zero_lemma.md`
   rules out the vanishing.

This is a valid conditional theorem, but it does not yet apply to the p24
relative sums.  The expression `Theta_a` selects the powers of a specific
order-`n` ideal class inside the horizontal CM torsor.  A seedless modular
realization of that selection either:

```text
uses the same embedded quotient/recovery data we are missing,
or keeps the full high-degree correspondence/branch structure.
```

This is the correspondence-degree obstruction recorded in
`correspondence_recurrence_resultant_boundary.md`.  Compactly writing the
class action does not lower the geometric pole degree unless a genuine
embedded class invariant has already been found.

For the third target, the degree mismatch is already visible:

```text
number of quotient fibers m = 66254,
relative fiber length n = 3107441.
```

The zero lemma would need a function with pole degree below `m` that evaluates
to `P_u(a)` on the `m` quotient fibers.  The natural `H`-cover realization
uses an order-`n` character projection and has pole degree at least on the
scale of `n`, already too large for this argument.  If one asks for an
`F_p`-rational Frobenius-packet norm, the root-of-unity packet degree

```text
ord_n(p) = 388430
```

inflates the natural degree scale to about

```text
n * ord_n(p) = 1207023307630.
```

So the zero lemma rules out bounded low-norm/local identities, but it does
not rule out the actual relative-resolvent collapse.

## Small-Scale Evidence

The natural CM relative scan now reports both the exact harmful condition and
the stronger individual-fiber/product condition.  A 100-case run:

```text
PYTHONDONTWRITEBYTECODE=1 PYTHONPATH=p24 \
python3 p24/natural_relative_resolvent_scan.py \
  --max-cases 100 --min-h 12 --max-h 96 --max-abs-D 20000 \
  --max-quotients 6 --q-stop 250000 --summary-only
```

found:

```text
rows=100
quotient_rows=317
relative_characters_tested=1268
relative_fibers_tested=4269
harmful_a_total=0
individual_zero_fiber_total=0
expected_random_zero_fibers=1.751551
all_equivalences_verified=1
```

Every quotient row also had

```text
min_nonzero_fibers = quotient_size,
```

so in this first 100-case scan the stronger product certificate happened to
hold: no individual relative fiber vanished.  The random-vector baseline for
this run predicted about `1.75` individual zero fibers, so the zero count was
encouraging but still small-sample evidence rather than a proof signal by
itself.

This supported the probability model, but it did not make individual-fiber
product nonvanishing a reliable theorem target.  It was also still not a p24
certificate, because the missing step was an explicit way to compute or prove
the selected high-order non-genus relative norm/content without enumerating
the embedded class set.

A broader natural scan later corrected the "product as theorem" hope:

```text
PYTHONDONTWRITEBYTECODE=1 PYTHONPATH=p24 python3 \
  p24/natural_relative_resolvent_scan.py \
  --max-cases 160 --min-h 12 --max-h 120 --max-abs-D 40000 \
  --max-quotients 6 --q-stop 400000 --summary-only
```

Output:

```text
rows=160
quotient_rows=520
relative_characters_tested=2150
relative_fibers_tested=7337
harmful_a_total=0
individual_zero_fiber_total=1
expected_random_zero_fibers=2.908526
all_equivalences_verified=1
```

The individual-zero row was:

```text
D=-1336, q=1777, ell=5, h=12, m=2, n=6.
```

Thus relative product nonvanishing is not a universal CM theorem.  It remains
a sufficient certificate if it can be proved for p24 specifically, but the
right general target is the exact relative-content nonvanishing or a packet
scalar such as Hermitian energy.

The product target is not the same as the group-determinant packet.  This is
recorded in:

```text
p24/relative_product_noninvariance_boundary.md
```

For fixed `a`,

```text
R_{a+r n} = sum_u zeta_h^(a u) zeta_m^(r u) P_u(a)
```

is an invertible Fourier transform.  This preserves the zero vector, but not
coordinatewise products.  Thus relative product nonvanishing and full
dual-coset resolvent product nonvanishing are two different sufficient
certificates.  The exact content certificate remains weaker than both.

## Current Consequence

The best constructive theorem target has split into two versions:

```text
exact:
  prove the relative content vector is nonzero for each of the eight p24
  Frobenius orbits;

stronger but cleaner:
  prove the quotient product/norm of the relative resolvents is nonzero for
  each of the eight p24 Frobenius orbits.
```

The second target might be accessible to a new Borcherds/Gross-Zagier-style
norm formula, but known formulas erase precisely the high-order non-genus
phase needed here.

The first-trace order-19 version is summarized in:

```text
p24/order19_relative_content_boundary.md
```

It is a cleaner small quotient for proving a future phase-aware identity, but
its recovery degree is too large to be the best certificate route.
