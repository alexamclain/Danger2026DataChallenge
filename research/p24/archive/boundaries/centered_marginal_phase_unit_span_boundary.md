# Centered Marginal Phase-Unit Span Boundary

Date: 2026-06-06

This note records a bounded recognition test for the simplest phase-aware
product formula for the centered determinant sequence.

## Question

Could the centered right determinant sequence

```text
Delta_C(t), t mod right,
```

be generated by elementary right-phase units such as

```text
Norm_{F_q(mu_right)/F_q}(1 - c*zeta_right^(k*t))
```

and constants?

If yes, the desired `Psi_O` in

```text
p24/centered_marginal_phase_borcherds_target.md
```

would have a concrete cyclotomic-unit product model.  If no, then a
successful phase-aware Borcherds/Fitting theorem must construct a more
specific Schubert determinant section.

## Audit

Added:

```text
p24/centered_marginal_phase_unit_span_scan.py
```

The scan:

```text
1. computes the centered normalized right determinant sequence in small
   actual-CM rows;
2. builds bounded right-binomial norm units, constants, and optionally
   small Heegner-fiber units;
3. converts all nonzero values to discrete-log vectors modulo each prime
   factor of q-1;
4. tests whether the target log vector lies in the dictionary span.
```

## Results

With only right-binomial units and constants, the three pinned actual-CM rows
miss outright:

```text
D=-6719, q=6863, pair=(3,7):
  q-1 factors: 2,47,73
  dictionary_rank=2/7 for every component
  target_in_span=0 for every component.

D=-13319, q=13463, pair=(4,7):
  q-1 factors: 2,53,127
  dictionary_rank=3/7 for every component
  target_in_span=0 for every component.

D=-10919, q=11243, pair=(3,13):
  q-1 factors: 2,7,11,73
  dictionary_rank=2/13 for every component
  target_in_span=0 for every component.
```

After porting the stronger trace-GCD special-divisor dictionary, the scan
also includes Heegner-fiber units:

```text
heegner-fiber:
  product over origins with fixed right phase of (j_origin - h),
  where h is a reduction of a small Heegner root.
```

Using `max_heegner_abs_D=500` and `max_heegner_h=10`:

```text
D=-6719, q=6863, pair=(3,7):
  unit_dictionary_size=82
  every prime component has dictionary_rank=7/7
  target_in_span=1
  first_containment_random_rate=1.0.

D=-13319, q=13463, pair=(4,7):
  unit_dictionary_size=97
  every prime component has dictionary_rank=7/7
  target_in_span=1
  first_containment_random_rate=1.0.

D=-10919, q=11243, pair=(3,13):
  unit_dictionary_size=76
  every prime component has dictionary_rank=13/13
  target_in_span=1.
  The only non-full-rank first containment is mod 2 at rank 12/13,
  with random containment rate 0.58.
```

So the richer dictionary does not give non-random recognition.  It reaches
full ambient rank quickly, and the target is then contained for the same
reason a random vector is contained.  The one non-full-rank prefix hit is too
weak and too mod-2-specific to suggest a product formula.

The holdout row `D=-26759`, `q=26903`, `m=21`, `n=11` separates a symmetric
artifact from the p24-like asymmetric behavior:

```text
pair=(7,7):
  target_values are constant;
  first containment has low random rate, likely because the sequence is
  constant rather than because a p24-like phase formula was found.

pair=(3,7):
  target_values=[1817,26744,13803,14969,9974,17858,21510];
  first containment occurs only at full rank 7/7 for both prime components;
  random containment rate is 1.0.
```

## Consequence

This demotes both easy versions of the special-divisor route:

```text
Delta_C(t) is not recognized as a product of small right-binomial norm units
and constants in the first actual-CM rows;

after adding small Heegner-fiber units, recognition appears only when the
dictionary spans the whole phase-vector space.
```

Thus the missing `Psi_O` is not visible in the bounded elementary/small
Heegner-fiber dictionary.  A scalar or full-rank vector fit is not enough; a
useful formula must identify the actual centered Schubert/Fitting divisor.

The surviving phase-aware p-unit theorem must therefore use one of:

```text
1. the actual Schubert/Fitting determinant section;
2. a CM-specific Borcherds/Siegel product beyond elementary right-binomial
   units;
3. a different normalization that changes the determinant sequence before
   applying a unit dictionary.
```

This does not disprove the phase-aware route.  It narrows it: a proof has to
construct the centered Schubert/Fitting object itself, not just recognize the
sequence as a small cyclotomic-unit product.
