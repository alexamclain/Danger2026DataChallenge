# Trace-GCD Chow Phase-Divisor Span Boundary

Date: 2026-06-06

This note records a bounded recognition test for a possible phase-aware
product formula for the trace-GCD Chow determinant.

The previous scans established:

```text
plain j / one-edge recognition: negative;
right-phase descent: positive in the pinned small row;
p24 exterior support: full by subset size 3.
```

The next possible shortcut is that the right-phase Chow sequence might be a
product of simple phase-aware units.  This would give a concrete candidate
for the desired `Psi_O`.

## Audit

The scan is:

```text
p24/trace_gcd_chow_phase_divisor_span_scan.py
```

It builds a bounded dictionary of unit-valued right-phase vectors:

```text
right-binomial:
  Norm_{F_q(mu_right)/F_q}(1 - c*zeta_right^(k*t));

heegner-fiber:
  product over non-right origins with fixed right phase of (j_i - h),
  where h is a small Heegner reduction root.
```

It converts those unit vectors and the Chow determinant vector to discrete
logs modulo the prime factors of `q-1`, and asks whether the Chow vector lies
in the span before the dictionary becomes the whole ambient space.

Command:

```text
PYTHONDONTWRITEBYTECODE=1 PYTHONPATH=p24 python3 \
  p24/trace_gcd_chow_phase_divisor_span_scan.py \
  --max-factor-degree 8 \
  --max-extension-degree 8 \
  --min-left-orbit-len 2 \
  --max-origin-shifts 140 \
  --max-binomial-constant 8 \
  --max-heegner-abs-D 500 \
  --max-heegner-h 10 \
  --random-controls 50
```

## Result

Pinned row:

```text
D=-13319
q=13463
h=140
m=28
n=5
pair=(4,7)
right=7
q-1 = 2 * 53 * 127
unit_dictionary_size=78
```

For both omitted Chow sequences:

```text
mod 2:
  dictionary_rank=3/7
  target_in_span=0

mod 53:
  dictionary_rank=7/7
  target_in_span=1
  random_containment_rate=1.000000

mod 127:
  dictionary_rank=7/7
  target_in_span=1
  random_containment_rate=1.000000
```

So the bounded dictionary does not give a non-random product formula:

```text
the only non-full-rank component misses the target;
the odd components contain the target only after spanning the full space.
```

## Consequence

This demotes the simplest phase-aware unit-product explanation.  It does not
disprove a bespoke Borcherds/Fitting section, but it says that the section is
not visibly generated by small cyclotomic phase units plus small Heegner
fiber units in this actual-CM row.

The proof target should therefore remain:

```text
construct the exact Schubert-Fitting/Chow section,
or prove its seven orbit norms are p-units directly.
```

A future positive product formula will need a more specific divisor
construction than the bounded unit dictionary tested here.

```text

```

It packages this bounded dictionary test with a determinant-zero control.
Default run:

```text
PYTHONDONTWRITEBYTECODE=1 PYTHONPATH=p24 python3 \
  p24/trace_gcd_phase_divisor_identity_holdout.py
```

reports:

```text

nonrandom_span_hits=0
full_rank_interpolation_hits=4

```

The determinant-zero control uses the same block size as the pinned actual-CM
row and has all entries and coordinate powers nonzero while the determinant is
zero.  Thus coordinate/Kummer-style payloads fail exactly the zero-detection
test that a true Fitting/Borcherds divisor identity would have to pass.
