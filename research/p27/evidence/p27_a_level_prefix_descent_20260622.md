# P27 A-Level Prefix Descent

Date: 2026-06-22

## Claim

The selected halving tower descends to whole `A` fibers much deeper than the
first conic sign-quotient screens showed.  On `12000 + 12000` p27
train/heldout samples, every selected gate d3..d14 had:

```text
mixed_A_groups = 0
mixed_A_rows = 0
```

This strengthens the conic/A-line moonshot: the live object is an A-level
sequence of Kummer characters.  It also keeps the main warning intact: the
number of surviving A groups still thins like independent half-loss, so this
is not a source-space shrink or a production GPU filter by itself.
