# P27 Alpha Branch-Class Screen

Date: 2026-06-21

## Claim

The alpha quotient is now geometrically explicit, but its visible branch
squareclass is already spent before the later selected-halving gates.

For the eliminated cyclic-quartic model over `E: W^2=X^3-X`,

```text
R^4 - 2*a*R^2 + b = 0
a = prefactor*m0
b = 4*prefactor^2*T2*S^2
```

the discriminant as a polynomial in `R` has squareclass:

```text
disc_R squareclass = T2
```

So the unique visible quadratic subcover of the alpha degree-4 cover is just
the already-known `T^2=T2` layer.  On every active p27/guard-field d3/d4
quotient row tested here, `T2` is already square.  Therefore the alpha branch
discriminant cannot be the missing post-domain / post-compactD selector.

## Branch Discriminant Screen

The branch-class probe explicitly records:

```text
disc_R squareclass = T2
unique_quadratic_subcover = T^2=T2
```

It then evaluates `T2` and the natural alpha branch atoms against descended
`d3` and `d4` rows.

On p27 train:

```text
d3 rows = 5000, T2_chi_1 = 5000
d4 rows = 2466, T2_chi_1 = 2466
d3 exact branch-atom combos = 0
d4 exact branch-atom combos = 0
d3 train best = 2572/5000 = 0.5144
d4 train best = 1312/2466 = 0.5320
```

On independent p27 heldout:

```text
d3 rows = 5000, T2_chi_1 = 5000
d4 rows = 2522, T2_chi_1 = 2522
d3 exact branch-atom combos = 0
d4 exact branch-atom combos = 0
d3 heldout best = 2640/5000 = 0.5280
d4 heldout best = 1332/2522 = 0.5282
```

The p27 train-best products do not replicate:

```text
d3 train-best on heldout = 2522/5000 = 0.5044
d4 train-best on heldout = 1280/2522 = 0.5075 for the best survivor,
                            with the main train-leading atoms at 0.4980
```

Guard fields:

```text
q1607: d3 T2_chi_1 = 196/196, d4 T2_chi_1 = 112/112, exact combos = 0/0
q1847: d3 T2_chi_1 = 252/252, d4 T2_chi_1 = 180/180, exact combos = 0/0
```

Some small-field best rates are large, but they are raw-bias artifacts: for
example `q1847_d3` and `q2087_d4` list `combo=1` among the best rows.

## Interpretation

Positive:

```text
The degree-4 alpha cover geometry is now sharply understood.
The projection ramification degree is verified by Magma.
The unique visible quadratic subcover is the T2 layer.
```

Negative:

```text
T2 is constant +1 on the active d3/d4 domains.
The natural alpha branch atoms do not give exact d3/d4 formulas.
The p27 train lifts collapse on heldout.
```

This kills the branch-discriminant version of the alpha/H90 route.  The alpha
cover remains useful as a coordinate system and prior-art explanation, but it
does not itself provide the post-compactD recurrence needed to beat sqrt.

## Consequence

The next sqrt-beating test should stop asking for more functions generated by
the alpha branch divisor.  It should extract the actual `d3` and `d4` double
covers on the residual elliptic quotient or its 2-isogenous/Kummer reduction:

```text
E:  W^2 = X^3 - X
E': V^2 = U^3 + 4U
K = x([2]P) on E'
```

A positive next artifact would be one of:

```text
explicit branch divisor / Kummer class for d3 over E' or P1_K
proof that d4 is the same class after a named map
low-genus source cover for the all-plus iterated 2-cover
GPU-testable direct sampler into that source
```

## Continue / Kill

```text
continue = exact d3/d4 cover extraction on E', P1_K, or reverse-source model
continue = Magma/Sage ramification/divisor-class computation for the d3 cover
continue = compare d3 and d4 only after d3 branch class is named

kill = alpha branch discriminant as a d3/d4 selector

```
