# P27 Label-2 Visible Residual-E Character Screen

Date: 2026-06-21

## Claim

The `compactD=-1` cyclic-quartic character does not collapse to a visible
quadratic character on the residual elliptic curve from the natural H90
factors tested here.

Across `1904` nondegenerate rows over small finite fields, no exact GF(2)
combination of the named residual-E factors matched the compactD bit.  The
best combinations hit only about `55.0%`, which is not a useful selector.

This is a finite-field diagnostic, not a proof.  It does, however, kill the
most obvious "cheap character on E" route.

## Screened Features

The tested factors were the natural divisor terms from the order-4 lift and
the label-2 equations:

```text
X
W
X-1
X+1
X^2+1
X^2+2X-1
X^2-2X-1
S = W(X+1)+2X^2
S_conj = W(X+1)-2X^2
mt_linear = 2WX + X^3 + X^2 - X - 1
m0
mt_coeff
prefactor = W(X^2+1)/X
L = 4WX^2 + 4WX + X^4 + 6X^3 - 2X - 1
```

Rows were restricted to residual elliptic points where:

```text
T^2 = X(X^2+1)(X^2+2X-1)
```

has an `F_p` root, the target compactD squareclass is nonzero, and all tested
feature characters are nonzero.

## Result

The T-sign invariance check passed throughout:

```text
t_sign_mismatch = 0 for every tested prime
```

This confirms the order-4/H90 result at the finite-field character level: the
compactD bit is independent of the choice of `T` sign when `T` exists.

But the visible character screen is negative:

```text
combined_rows = 1904
combined_exact_combo = none
```

Best in-sample combinations:

```text
good=1048/1904 rate=0.550420168 combo=X * W * S_conj
good=1048/1904 rate=0.550420168 combo=X2+2X-1 * X2-2X-1 * mt_linear
good=1048/1904 rate=0.550420168 combo=S * S_conj * mt_linear
good=1048/1904 rate=0.550420168 combo=X2+1 * S_conj * prefactor
good=1048/1904 rate=0.550420168 combo=X2-2X-1 * m0 * prefactor
good=1048/1904 rate=0.550420168 combo=W * S_conj * L
```

Several very small primes had accidental exact formulas on tiny row counts,
but these do not persist in the combined screen.

## Interpretation

Positive:

```text
The compactD bit really is T-sign invariant.
The order-4 lift survives a broader finite-field character check.
```

Negative:

```text
The bit is not a visible quadratic character generated by the natural residual
E factors above.
No screened factor gives a practical selector or source.
```
