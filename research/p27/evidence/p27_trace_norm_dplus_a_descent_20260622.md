# P27 Trace/Norm Dplus A-Descent Bridge

Date: 2026-06-22

## Claim

Post-Dplus selected gates in the trace/norm raw-y stream descend to whole
`A` fibers in the same way as the conic/A-line lane.

On three independent p27 seed groups, after conditioning on trace/norm
`Dplus`, every tested `A` group has a constant `d3` sign, and every `A` group
on the `d3=+1` slice has a constant `d4` sign:

```text
d3_A_mixed_rate = 0
d4_A_after_d3_mixed_rate = 0
```

This bridges two live fronts.  It says that Dplus post-gate coupling should
be routed into A-level Kummer/divisor class extraction, rather than treated as
a separate trace/norm bucket-search lane.  It does not by itself give a
sampler or a source-space shrink.

## Interpretation

Positive:

```text
Dplus post-gates are A-level in these p27 streams.
The trace/norm and conic/A-line fronts share the same later-gate class surface.
GPU Dplus telemetry should emit A and selected bits, not only trace/norm signs.
The A-level Kummer extraction packet is now the right destination for Dplus
post-gate class comparison.
```

Negative:

```text
The A-level signs remain balanced.
This is not a source sampler.
This does not rescue H90 payload sign buckets; those were already heldout-flat.
Dplus remains an exact two-gate prefix, not a below-sqrt mechanism unless the
A-level class sequence recurs or becomes sourceable.
```

## Consequence

Update the Dplus lane priority:

```text
keep = fused/native Dplus pricing as an engineering exchange-rate test
keep = Dplus telemetry columns if they emit A and later selected bits
route = post-Dplus d3/d4 class work to A-level Kummer extraction
kill = separate low-weight trace/norm sign buckets for post-Dplus d3/d4
kill = treating Dplus-H90 payload signs as the independent later-gate moonshot
```

The next mathematical object remains:

```text
normalized A-line Kummer/divisor classes for d3, d4, d5, ...
```

Promote only if those classes are pullbacks, translates, coboundaries,
iterates, or otherwise generated by a sourceable low-genus structure that
controls multiple selected gates.
